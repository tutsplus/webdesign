CSS padding Property

The padding property in CSS sets the padding area on all four sides of an element's content box. It creates space around the content within the element and affects the layout and spacing of other elements around it.

Syntax

The padding property can be specified using one of the following syntax forms:

  • Four values: padding: top right bottom left;
  • Three values: padding: top horizontal bottom;
  • Two values: padding: vertical horizontal;
  • One value: padding: value;

Each value can be expressed as an absolute or relative length, or using a percentage. Negative values are not allowed.

Values

  • length: Specifies the padding as a fixed length, such as pixels (px), points (pt), or em units (em).
  • %: Specifies the padding as a percentage of the width of the containing block.

Examples

/* Applying the same padding to all sides */
.box {
  padding: 20px;
}

/* Applying different padding to each side */
.box {
  padding: 10px 20px 15px 30px;
}

/* Applying vertical and horizontal padding */
.box {
  padding: 10px 20px;
}

Remarks

  • The padding property applies to all types of elements, including block-level and inline-level elements.
  • If the box-sizing property is set to border-box, the specified padding will be included in the total width and height of the element.
  • Padding values contribute to the total dimensions of an element and may affect its position within the layout.
  • The order of specifying values follows the clockwise direction, starting from the top.
  • If fewer than four values are specified, they will be applied to the sides in the order top, right, bottom, and left.

Compatibility

The padding property is supported in all modern web browsers. It is also backward compatible with older browsers.

For detailed information on browser compatibility and specific vendor prefixes, refer to the MDN web documentation on padding.
